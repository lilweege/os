# os

A toy 32-bit operating system for x86

## Quick Start
Requirements:
- GNU coreutils (`make`, `cat`, `dd`)
- GNU gcc for i386 (`i386-elf-gcc`)
- NASM (`nasm`)
- QEMU (`qemu-system-i386`)


```sh
$ make run
```
